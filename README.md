## Getting Started

Flutter version: 3.24.0
Flutter manager used: puro

## How to Run

To



To run the **Money Send App** project, follow these steps:

1. Install the required dependencies:

   ```bash
   flutter pub get
   ```

2. Start the app:

   ```bash
   flutter run
   ```

   This will launch the app on the connected device or emulator.
